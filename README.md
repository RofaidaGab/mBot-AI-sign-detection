You start this by running the python file after uploading arduino code to the mbot. C-ROAD AND C-ROAD END SIGNS WERE NOT USED. The ai model still detects them, but they werent used and the car relies solely on fixed diameter values 30 and 100 for turns and on line following sensors! You may adjust the code to your liking for using c-road signs. the mbotcomm arduino file is where the esp32 camera was programmed to give the IP address each time my hotspot was connected. NOTE: IP address may vary according to place but will 100% vary according to hotspot. make sure the hotspot is also connected to your laptop. The same code also sets up tx rx communication. Python code runs the ai model on the stream using the ip address then sends commands back to the camera through wifi then camera sends them to arduino by tx rx.
